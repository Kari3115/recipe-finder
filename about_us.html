Section 2 - Project Description and why you chose this project
This is a Recipe Finder website which has a search feature that helps users look up different recipes whenever they are tired of daily tedious meals. The intention of building this website is that I am a big foodie and sometimes rely on Youtube videos to make something special. So I feel this project will be pretty helpful to the users who are foodies like me and open to experiment. Also, there is an option to create, modify and delete the recipes which have been added by the existing members and only the members can create or make changes to those recipes.

Section 3 - Description of technology used for your site
Recipe Finder is a simple website built using HTML, CSS, Ajax and Javascript that uses Amazon DynamoDB as database. It has navigation links to multiple pages which give a good overall understanding of the website. GitHub Pages has been used to host it.

Section 4 - What have you learned in this class

As the curriculum was very well-structured, I got to learn a wide scope of skills and gained a good amount of hands-on experience while completing the assignments. Overall, I would say that this course was very useful in providing basic understanding and skills that go behind building a full-fledged website.
